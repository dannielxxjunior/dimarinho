      DiMarinho Catálogo online

Este projeto consiste em uma especie de catálogo online que decidi criar para que a minha mãe de quem estou afastado a 11 anos podesse partilhar o seu trabalho com o Mundo.
Para mim foi como juntar o melhor de dois mundos, o meu aprendizado, permitindo-me praticar e desenvolver meus conhecimentos, e oferecer também algo unico e inesperado a alguém que seria capas de dar seu proprio mundo por mim.
Existe espaço para bastantes melhorias, caso queiras fica a vontade para dar a tua opinão, e quem sabe contribuir para um projeto que foi criado apenas por amor.
